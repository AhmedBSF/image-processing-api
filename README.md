## Getting Started

### Running

+ `npm install`
+ `npm run start`

### Test build

+ `npm run build`

### Lint & prettier

+ `npm run lint`
+ `npm run prettier`

### Run tests

+ `npm run test`

## Folder structure

```bash
 .
 ├── spec                    # Unit tests configuration
 ├── public                  # Assets (Images)
 ├── src                     # Source files
 │   ├── routes
 │   ├── utilities
 ├── tests                   # Unit tests
 └── README.md
```;
